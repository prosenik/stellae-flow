# Changelog

## [1.0.0] — 2026-02-18

### Added
- **Prototype Connection Reader** — Scans all prototype connections on the current page
- **Flow Diagram Generator** — Creates clean flow diagrams with screen thumbnails and labeled arrows
- **Auto-Layout** — dagre.js-powered intelligent graph layout (LTR and top-down)
- **Flow Highlighting** — Color-coded paths for different user journeys (Pro)
- **Interaction Labels** — Display trigger types on connection arrows (Pro)
- **PNG Export** — Export flow diagrams as PNG images (Free)
- **PDF Export** — Export flow diagrams as PDF documents (Pro)
- **Free & Pro tiers** — Free supports up to 10 screens; Pro unlocks unlimited screens and advanced features
